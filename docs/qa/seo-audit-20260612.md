## Critical Findings

### P1 — High Priority

#### 1. Missing OpenGraph tags on ALL pages
No page exports `og:image`, `og:url`, or `og:type`. Social media shares (Facebook, Zalo — dominant in Vietnam) display generic previews. Root layout has `openGraph.locale: 'vi_VN'` and `siteName` but no page-specific OG.
**Fix:** Add per-page OG in `generateMetadata()` — especially `/trips/[id]` (shareable booking link).

#### 2. Incomplete metadata on `/search` — highest-traffic page
Title "Tìm chuyến xe" (21ch) is vague. Description "Tìm và đặt vé xe khách" (24ch) is far below 140-160ch target. No OG tags. No SearchResultsPage schema.
**Fix:** Dynamic title: `Xe {origin} đi {destination} — Đặt vé online | BBVN`. Description: include operator count, price range, frequency.

#### 3. Hero images use CSS `backgroundImage` — no alt text, no next/image
Home page hero occupies 30%+ of viewport. Uses inline `backgroundImage` CSS — invisible to search engines, no alt text, no srcset optimization. Comment in code acknowledges: "Production LCP upgrade: replace with `<Image fill priority>`".
**Fix:** Replace with `<Image>` component, add descriptive alt in Vietnamese.

### P2 — Medium Priority

#### 4. Weak page titles across all pages
All titles < 26 characters. Missing primary keywords: "liên tỉnh" (intercity), "online", "giá rẻ" (cheap). Vietnamese bus-booking SERPs are competitive.
**Fix:**
- Home: `Đặt vé xe khách liên tỉnh online — Nhanh, An toàn | BBVN` (52ch)
- Search: `Xe {origin} đi {destination} — Đặt vé online | BBVN` (dynamic)
- Routes: `Tuyến xe khách liên tỉnh — Lịch trình & Giá vé | BBVN` (48ch)

#### 5. Generic descriptions — lack action-oriented language
Home description is 52 characters (target: 140-160). No call-to-action, no differentiators.
**Fix:** `Đặt vé xe khách liên tỉnh an toàn, nhanh chóng trong 30 giây. Hàng trăm chuyến mỗi ngày, giá tốt nhất, đặt online dễ dàng.` (118ch)

#### 6. No structured data on `/routes` page
Route directory has no ItemList or BreadcrumbList schema. Missing SEO opportunity for route-specific rich results.
**Fix:** Add `ItemList` JSON-LD with route items.

#### 7. Search results missing SearchResultsPage schema
Trip results are rendered but not structured for Google. No ItemList wrapping individual BusTrip items.

#### 8. Trip detail missing AggregateRating & availability
BusTrip schema is present but lacks `availableSeats`, `priceRange` for rich snippets.

### P3 — Low Priority

#### 9. No related trips/routes suggestions on trip detail
Internal linking opportunity missed — "Các chuyến khác từ {origin}" section.

#### 10. `/terms` and `/privacy` listed in sitemap but may not exist as pages

#### 11. Charter page missing BreadcrumbList JSON-LD
